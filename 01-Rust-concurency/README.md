# Проверка чисел на простоту с использованием многопоточности (Rust)

Этот проект представляет собой многопоточное консольное приложение на языке **Rust**, предназначенное для параллельной проверки чисел на простоту.  
Основной акцент программы — эффективное распараллеливание вычислений и использование всех доступных ядер процессора.

---

## Основная идея

Проверка чисел на простоту — задача, которая хорошо масштабируется по потокам:  
каждое число обрабатывается независимо, поэтому нагрузку можно легко распределить между несколькими ядрами процессора.

В данной реализации используются:

- автоматическое определение числа доступных CPU-ядер;
- разбиение списка чисел на блоки (chunks);
- запуск нескольких потоков (`std::thread::spawn`);
- передача результатов через канал (`mpsc::channel`);
- финальная сортировка для восстановления исходного порядка чисел.

---

## Формат входных данных

Программа принимает текстовый файл, где:

- каждая строка содержит одно целое число;
- допустимы пустые строки и пробелы;
- некорректные строки автоматически игнорируются.


### Пример "input.txt":
5
7
8

##  Формат выходных данных
Для каждого числа программа выводит **ровно одну строку** результата.(prime / not prime)
### Пример:
5 is prime
7 is prime
8 is not prime
