# Описание программы
Эта программа на языке D конвертирует Markdown-разметку, хранящуюся в обычном текстовом файле *.txt, в HTML-документ.
# Формат входного файла
Хотя файл имеет расширение .txt, его содержимое — это Markdown-разметка.
Пример входного файла, создаваемого программой:
<pre>
# Hello World

This is a test.

## List

- Milk
- Bread
- Eggs

Normal text again.
</pre>
