# CSV → XML Converter (Standard ML)

## Описание проекта

Данная программа реализует преобразование данных из формата **CSV** в **XML**
с поддержкой **схемы данных (schema)** и **сбором статистики**.

Программа написана на языке **Standard ML (SML/NJ)** и демонстрирует
использование:
- статической типизации,
- функционального стиля программирования,
- обработки файлов,
- валидации входных данных,
- агрегации статистики с помощью свёрток (`fold`).

---

## Входные данные (Input)

### Формат CSV-файла

Программа принимает **один CSV-файл**.

### Первая строка — схема данных

Формат: <имя_поля>:<тип>,<имя_поля>:<тип>,...
Поддерживаемые типы:
- `int`
- `real`
- `string`
### Последующие строки — данные

- Каждая строка — одна запись
- Значения разделены запятыми `,`
- Количество столбцов должно совпадать со схемой

### Пример `input.csv`

```csv
id:int,name:string,score:real
1,Alice,8.5
2,Bob,7.2
3,Charlie,9.1
