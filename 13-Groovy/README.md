# Описание программы
Эта программа на языке Groovy строит фрактал множества Мандельброта и выводит его в виде ASCII-графики прямо в консоль.
Все параметры рендеринга — ширина, высота и максимальное число итераций — считываются из простого текстового файла input.txt.
Программа использует:
- численный расчёт на комплексной плоскости;
- рекурсивную функцию для проверки выхода за пределы множества;
- ASCII-палитру для визуализации глубины итераций.
# Формат входного файла
Файл должен содержать 3 строки:
<pre>
width — ширина ASCII-изображения
height — высота
maxIter — максимальное число итераций для проверки точки
</pre>
Например:
<pre>
120
40
1000
</pre>
