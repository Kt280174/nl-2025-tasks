# Описание
Данный код на Julia предназначен для извлечения адресов электронной почты и номеров телефонов из входного текстового файла.
# Формат входных данных
## Входной файл:**input.txt**
Содержит произвольный текст, который может включать email-адреса и номера телефонов.\
Пример содержимого:
<pre>
Контакты: john.doe@example.com или support@company.org
Телефон: +7-495-123-45-67 или 812 345 67 89</pre>
# Формат вывода
Результаты выводятся в консоль в следующем формате:
<pre>
Emails found:
email1@example.com
email2@domain.org
(none)  # Если ничего не найдено

Phones found:
+7-495-123-45-67
812 345 67 89
(none)  # Если ничего не найдено
</pre>
