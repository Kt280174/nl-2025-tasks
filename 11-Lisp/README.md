# Клиент–серверный проект (Python + Common Lisp)
Этот проект представляет собой простую систему клиент–сервер:
- сервер написан на Common Lisp
- клиент написан на Python
- клиент читает команды из файла, отправляет их серверу
- ответы сервера сохраняются в responses.txt
Подходит для демонстрации socket-взаимодействия и текстового протокола.
# Формат входных команд (requests.txt)
Каждая команда — это отдельная строка.
Поддерживаемые команды:
| Команда     | Описание                  |
| ----------- | ------------------------- |
| `SUM a b c` | сумма всех чисел          |
| `REVERSE s` | строка в обратном порядке |
| `UPPER s`   | перевод в верхний регистр |
| `LOWER s`   | перевод в нижний регистр  |
| `QUIT`      | завершение соединения     |

# Формат выходного файла (responses.txt)
Формат строки:
<pre>
<команда> ==> <ответ>
</pre>
