# Описание программы
Программа на OCaml считывает лабиринт из текстового файла, ищет путь от точки S до точки E с помощью алгоритма DFS (поиск в глубину) и выводит лабиринт с отмеченным маршрутом.
# Формат входного файла (input)
Файл **input.txt** должен содержать лабиринт из символов:\
Пример:
<pre> 
####### 
#S....# 
#.#.#.# 
#.#E#.# 
#.....# 
#######</pre>

# Формат выходных данных
Если путь найден — программа выводит лабиринт, где маршрут отмечен *.

Пример вывода:

<pre> 
Path found:
#######
#S**..#
#.#*#.#
#.#E#.#
#.....#
#######</pre>
