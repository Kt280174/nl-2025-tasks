# Интерпретатор простого стекового языка (Ruby)
Этот проект реализует мини-интерпретатор пользовательского языка, основанного на стеке. Интерпретатор читает команды из файла **input.txt**, выполняет их и выводит результат.
Язык поддерживает арифметику, переменные, условные операторы IF, циклы LOOP, а также стандартные стековые команды.
# Формат входного файла (input.txt)
Каждая строка содержит одну команду языка.
Поддерживаемые команды:
| Команда | Описание |
|--------|----------|
| SET x 10 | присвоение переменной |
| PUSH x   | положить значение в стек |
| ADD      | сложить два значения из стека |
| PRINT    | вывести верх стека |

Пример входа
<pre>
SET A 5
SET B 3
PUSH A
PUSH B
ADD
PRINT
</pre>
