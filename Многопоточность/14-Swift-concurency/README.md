# Анализ логов с многопоточностью на Swift
Этот проект представляет собой многопоточный анализатор логов, написанный на Swift.
Программа читает лог-файл, разбирает каждую строку, распределяет данные на несколько потоков и собирает статистику:
- количество записей по уровню (INFO, ERROR, …)
- количество записей по модулю
- активность по часам
- топ повторяющихся сообщений
# Формат входного файла
Каждая строка должна соответствовать формату:
<pre>
[YYYY-MM-DD HH:MM:SS] [LEVEL] [MODULE] message text...
</pre>
