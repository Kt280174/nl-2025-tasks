# Интерпретатор полиномов на Haskell
Данная программа на Haskell выполняет разбор многочленов (полиномов), представленных в виде строк, и осуществляет над ними различные операции:
сложение, умножение, вычисление производной и вычисление значения в точке.
# Формат входного файла
Файл должен содержать минимум 3 строки:
<pre>
<полином P(x)>
<полином Q(x)>
<целое число x0>
</pre>
# Формат вывода
Программа печатает:
<pre>
P(x) = 
  <нормализованный полином P>

Q(x) =
  <нормализованный полином Q>

P(x) + Q(x) =
  <сумма>

P(x) * Q(x) =
  <произведение>

P'(x) =
  <производная P>

Q'(x) =
  <производная Q>

P(x0) = <число>
Q(x0) = <число>
(P+Q)(x0) = <число>
</pre>
