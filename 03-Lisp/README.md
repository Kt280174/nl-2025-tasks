# Конвертация CSV → XML на языке Lisp
Данный проект реализует простую систему преобразования данных:
входной файл CSV → выходной файл XML.
Решение выполнено на чистом Common Lisp без сторонних библиотек.
# 1. Описание задачи
Цель лабораторной работы — показать:
- работу с файлами в Lisp;
- построчное чтение и разбор CSV;
- обработку строк и разбиение на поля;
- генерацию XML-документа в соответствии со структурой данных;
- реализацию преобразования в функциональном стиле
# 2. Формат входного файла(CSV)
Ожидается обычный CSV с первой строкой — заголовками, например:
<pre>
name,age,city
Ivan,23,Moscow
Anna,20,Kazan
Petr,31,Sochi
</pre>
Требования:
- значения разделены запятой;
- кавычки не используются (упрощённый CSV);
- каждая строка соответствует отдельному объекту.
