# Simple TCP Server & Client (Elixir)

Данный проект демонстрирует работу простого TCP-сервера и клиента на языке **Elixir**.  
Сервер принимает строковые сообщения от клиента, обрабатывает их и отправляет обратно ответ вида:
RECEIVED: <message>
Клиент читает строки из входного файла, отправляет их серверу и выводит полученные ответы.

---

## Архитектура программы

Проект состоит из двух основных модулей:

### 1. `SimpleServer`
Отвечает за:
- запуск TCP-сервера на указанном порту,
- приём клиентских подключений,
- обработку входящих сообщений,
- отправку ответа клиенту.

Используются функции:
- `:gen_tcp.listen/2`
- `:gen_tcp.accept/1`
- `:gen_tcp.recv/2`
- `:gen_tcp.send/2`

Каждое подключение обрабатывается в отдельной задаче (`Task.start/1`), что делает сервер многопоточным.

### 2. `Client`
Отвечает за:
- подключение к серверу,
- отправку строк по TCP,
- получение ответа от сервера,
- вывод обмена на экран.

---

## Формат входных данных

Файл **`input.txt`**:

Каждая строка — это отдельное сообщение, отправляемое серверу.

### Пример input.txt:
Hello
How are you?
Elixir is great!
## Формат выходных данных

Клиент выводит:

1. Строку, которую он отправил.
2. Ответ сервера в формате:
RECEIVED: <исходная_строка>

### Пример вывода:
Client: Hello
Server: RECEIVED: Hello

Client: How are you?
Server: RECEIVED: How are you?

Client: Elixir is great!
Server: RECEIVED: Elixir is great!
