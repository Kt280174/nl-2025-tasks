# Асинхронный Web-Scraper на F#
Этот проект на F# реализует простой, но очень наглядный веб-скрепер, который:
- считывает список URL-адресов из файла urls.txt;
- асинхронно загружает каждую страницу с помощью HttpClient;
- извлекает HTML-тег <title>;
- выводит результат для каждого сайта;
- корректно имитирует настоящий браузер с помощью поддельного User-Agent.
Все запросы выполняются параллельно благодаря Task и Task.WhenAll.
# Формат входного файла
Каждая строка — один URL.
Пустые строки автоматически игнорируются.
Пример:
<pre>
https://example.com
https://www.wikipedia.org
</pre>
